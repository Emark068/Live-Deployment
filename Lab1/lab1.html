This is a simple paragraph explaining front-end basics.


Front-end development is the art of building everything a user directly sees and interacts with on a website,
acting as the bridge between design and functionality. The foundation of this process relies on three
core technologies: HTML (HyperText Markup Language) provides the structure, acting as the skeleton that organizes
text, images, and buttons. CSS (Cascading Style Sheets) provides the styling, acting as the skin and clothes that
determine colors, layouts, and responsiveness across different devices. Finally, JavaScript brings the website to life, acting as the brain to add interactivity, such as animations, form validation, and dynamic updates without needing to reload the page.
Together, these technologies ensure a user-friendly, visually appealing, and engaging
